# Lazy Loading com JavaScript

Projeto simples para praticar lazy loading de imagens com JavaScript.

A ideia foi simular um cenário onde a página tem muitas imagens e precisa carregar bem mesmo com internet mais lenta. Em vez de carregar tudo de uma vez, as imagens vão sendo carregadas conforme o usuário rola a página.

## Como funciona

As imagens começam em baixa resolução. Quando entram na área visível da tela, o JavaScript detecta isso e troca para uma versão em maior qualidade.

## Tecnologias

HTML, CSS e JavaScript

## O que pratiquei

- IntersectionObserver
- Manipulação do DOM
- Troca dinâmica de imagens
- Noção de performance no front-end
